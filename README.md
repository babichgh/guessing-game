# guessing-game
## Prerequisites
- [rustup](https://rust-lang.org/tools/install/)
## Installation
1. Clone the repository:
   ```
   git clone https://github.com/babichgh/guessing-game.git
   cd guessing-game
   ```
2. Run
   ```
   cargo run
   ```
## Project structure
```
file-writer/
```
